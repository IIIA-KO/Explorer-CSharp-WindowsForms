директориях с использованием элементов управления ListView и 
TreeView: отображение файлов и папок всех логических дисков с

■ при двойном клике на файле – открывать его стандартными средствами;

■ переименовать файл;
■ переименовать папку;
■ удалять файл;
■ удалять папку
